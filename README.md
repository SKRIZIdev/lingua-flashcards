# 🗣️ Lingua — Language Flashcards

A tiny flip-card vocabulary trainer for Spanish, French and German. Flip a card, mark it *Know it* or *Again*; unknown cards cycle back until learned. Progress per deck is saved in `localStorage`.

**Live demo:** https://skrizidev.github.io/lingua-flashcards/

## Pages
- **Study** — 3D flip cards with a deck picker and grading
- **Decks** — per-deck progress bars and a reset
- **About** — how to study

## Tech
Vanilla HTML / CSS / JavaScript with a CSS 3D card flip.
